
---
1. Launch server utilities:
```
docker-compose up -d
```
2. Launch backend:

```
npm i --save /yarn
```

```
yarn build
```

```
yarn start:prod
```
3. Launch frontend:

```
yarn
```

```
yarn build
```
Run in a daemon process or Docker container.
